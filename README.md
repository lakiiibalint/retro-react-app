# retro-react-app

Retro React App is a frontend application focused on retrospectives and project management. During the assignment, I was allocated to this project as a frontend developer intern, and the goal was to deliver a modern, modular React-based interface.

## What does the app do?

- Lists projects and shows project details.
- Manages the retro board with columns, notes, and action items.
- Displays users and role information.
- Provides login and registration screens.
- Supports multiple languages.
- Includes drag-and-drop based retrospective interactions.
- Uses form handling and validation.

## Tech stack

- React
- TypeScript
- Vite
- React Router
- Redux Toolkit
- TanStack Query
- React Hook Form
- i18next
- dnd-kit
- Axios
- Yup
- Tailwind CSS v4

## Running the project

The project cannot currently be run end-to-end because I was not given the required Docker image. As a result, the backend service and the related environment setup are not available.

If the Docker image and backend environment become available, the frontend can be started with:

```bash
npm install
npm run dev
```

## Note

The frontend part of the repository is complete, but the infrastructure required for a real runtime is missing. Because of that, the project is currently more suitable for development and demonstration purposes.
